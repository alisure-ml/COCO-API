# 安装

1. 下载
```
# old
git clone https://github.com/cocodataset/cocoapi.git
cd cocoapi/PythonAPI
```

```
# new
git clone https://github.com/nightrome/cocostuffapi.git
cd cocoapi/PythonAPI
```

2. 编译
```
# linux
make
```

```
# win
python setup.py install
# python setup.py build_ext --inplace
```
